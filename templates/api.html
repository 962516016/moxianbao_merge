参数详情

SDK客户端主要包括processing_data、save_predict_file、ai_analysis三大模块，分别用于数据分析和预处理、模型训练预测及可视化、AI智能分析。

processing_data模块

参数名称 是否必选 类型 说明
csv_file 是 str 通过该参数选择数据集路径
fill 否 bool 通过该参数选择是否对0值进行重新填充
deduplication 否 bool 通过该参数选择是否进行数据去重
outlier 否 bool 通过该参数选择是否对离群点检测并修正
normal 否 bool 通过该参数选择是否对数据进行归一化处理
resample 否 bool 通过该参数选择是否对缺失时间点的数据进行重采样

save_predict_file模块

参数名称 是否必选 类型 说明
csv_file 是 str 通过该参数选择数据集路径
figure 否 bool 通过该参数选择是否导出可视化图片
file 否 bool 通过该参数选择是否导出预测后的表格文件
train 否 bool 通过该参数选择训练并预测/预训练模型直接预测

ai_analysis模块

参数名称 是否必选 类型 说明
csv_file 是 str 通过该参数选择数据集路径
ai 否 bool 通过该参数选择是否对结果进行AI智能分析

使用方法简介

windpowerapi是风电功率预测与分析的Python SDK客户端，为使用风电功率预测与分析的开发人员提供了一系列的交互方法。 参考如下代码
新建一个windpowerapi.py： 

                            
 import windpower_upc as wp

 api_url = 'http://localhost:5446/windpowerapi'
 api_key = 'your API_KEY'
 filepath = '数据集路径'
 api = wp.windpowerapi(api_url, api_key)

 # 对数据集进行预测并保存结果文件
 api.savepredictfile(filepath, figure=False, file=True, train=False)
 api.savepredictfile(filepath, figure=True, file=False, train=False)

 # 对数据集进行预测并保存处理后的结果文件
 api.processingdata(filepath, fill=True, outlier=True, normal=False, resample=True, deduplication=True)

 # 对预测的数据集进行AI分析
 api.aianalysis(csv_file='predictions.csv', ai=True)

在上面代码中，API_KEY是系统分配给用户的，为字符串，用于标识用户，为访问做签名验证，可在个人中心查看。
